# Cloudforce_hackathon
🛰️ 
A production-ready AI agent that bridges the gap between educational content and accountability.

## 🚀 The Vision
Most AI tools just summarize text. **This tool** audits it. By analyzing YouTube transcripts, we provide:
- **Academic Scaffolding:** Multi-level summaries (Quick, Deep, and Full).
- **Sociotechnical Auditing:** Evaluating equity, accessibility, and epistemic bias in academic discourse.
- **Fact-Checking & Citations:** A Citation Agent that provides exact transcript quotes to verify claims.

---

## 🛠️ Tech Stack
- **Frontend:** React / Tailwind CSS (Deployed on Vercel)
- **Backend:** Python / FastAPI (Deployed on Render)
- **LLM Engine:** Google Gemini 2.0 Flash Lite
- **Data Extraction:** TranscriptAPI (Advanced Proxy Layer to bypass Datacenter IP blocks)
- **Infrastructure:** In-memory Caching & Cron Heartbeat for 100% uptime.
